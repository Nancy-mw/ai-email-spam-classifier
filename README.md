# ai-email-spam-classifier
# 📧 AI-Based Email Spam Classifier

## 📌 Overview

A machine learning model that classifies emails as spam or ham using natural language processing techniques.

## 🎯 Objective

To improve email filtering and detect spam messages accurately.

## 📊 Dataset

* Public spam email datasets(spam-ham dataset)

## 🛠️ Tools & Technologies

* Python
* Scikit-learn
* NLP (TF-IDF, Count Vectorizer)
* Jupyter Notebook

## ⚙️ Methodology

1. Text preprocessing (tokenization, cleaning)
2. EDA.
3. Feature extraction (TF-IDF)
4. Model training (Naive Bayes / Logistic Regression)
5. Model evaluation (accuracy, precision, recall)

## 📈 Results

* High classification accuracy.
* Efficient spam detection model.
